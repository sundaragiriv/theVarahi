# Varahi Media Prompt Pack

This site is ready for three primary motion assets. Generate them offline and drop them into `public/videos/`.

Recommended replacements:

- Home hero: `public/videos/video1.mp4`
- Proof / case studies: `public/videos/video2.mp4`
- Menu panel backdrop: either reuse `video2.mp4` for now or create a dedicated third file and wire it in next

General direction for every clip:

- Premium enterprise technology brand
- Slow, fluid, expensive motion
- No people
- No text
- No logos
- No dashboards
- No obvious stock-footage feeling
- No aggressive particles or sci-fi clutter
- Aspect ratio `16:9`
- Ideal duration `8-12 seconds`
- Seamless loop preferred

## 1. Home Hero

Use this for the homepage flagship motion layer.

Target feeling:

- Varahi as an intelligent systems brand
- Architectural, precise, premium
- More alive than static blocks, but still restrained

Prompt:

```text
Create a seamless 10-second cinematic loop for the homepage of a premium intelligent-systems company called Varahi. The scene should feel architectural, abstract, and enterprise-grade rather than futuristic for its own sake. Show vertically shifting modular forms, layered glass surfaces, soft grid structures, luminous scan lines, and controlled streams of cool blue light moving through the environment. Motion should be fluid, calm, and high-end, with subtle vertical drift and depth, like a living technology system. No people, no text, no logos, no dashboards, no obvious stock footage. Color palette: midnight navy, graphite, pale ice blue, silver-blue, soft white highlights. Camera movement minimal and elegant. Seamless loop. Aspect ratio 16:9.
```

## 2. Proof / Case Studies

Use this for the proof layer and case-studies page.

Target feeling:

- Operational clarity
- Systems coordination
- Quiet confidence

Prompt:

```text
Create a seamless 8-second cinematic loop for an enterprise proof and case-studies page. The visual should represent coordinated systems, process clarity, and intelligent operations. Show connected transparent panels, refined architectural forms, slow-moving light paths, subtle node connections, and layered depth that suggests control and execution. The motion should feel premium, restrained, and trustworthy. No people, no text, no logos, no dashboards, no charts. Color palette: deep navy, steel blue, slate, silver-blue, cool white accents. Camera motion minimal and smooth. Seamless loop. Aspect ratio 16:9.
```

## 3. Menu Panel Backdrop

Use this behind the light menu sheet so the menu feels elevated and cinematic instead of static.

Target feeling:

- Spatial depth
- Calm intelligence
- Elegant navigation layer

Prompt:

```text
Create a seamless 10-second loop for a premium navigation overlay on a technology brand website. Scene: minimal spatial corridor with frosted surfaces, layered architectural planes, soft reflections, luminous edge lighting, and slow drifting blue-white highlights. The visual should feel intelligent, clean, and expensive, with very smooth motion and subtle atmospheric depth. No people, no text, no logos, no hard sci-fi effects, no fast motion. Color palette: deep slate, soft white, cool stone, pale electric blue, muted cobalt. Seamless loop. Aspect ratio 16:9.
```
