# Video Assets

## Required Video File
Place your hero background video file here as `tech-background.mp4`

### Recommended Video Specifications:
- **Format**: MP4 (H.264 codec)
- **Resolution**: 1920x1080 or 4K
- **Duration**: 15-30 seconds (will loop seamlessly)
- **Size**: Under 10MB for optimal loading
- **Content**: Tech/futuristic theme (circuit boards, data streams, abstract tech visuals)

### Fallback Behavior:
If no video is present, the component will show a beautiful animated background with the space/tech image from Unsplash as a poster.

### Sample Video Sources:
- Pixabay: https://pixabay.com/videos/
- Pexels: https://www.pexels.com/videos/
- Unsplash: https://unsplash.com/videos/

### Current Fallback:
The hero section uses a high-quality space/tech image as a poster and fallback background with animated gradients.
